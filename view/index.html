Consola o Terminal

¿Que es?

Es la forma mas flexible en la que una persona puede interactuar con una computadora por medio de comandos entedibles para la maquina

En ocaciones es bueno personalizar las consola ya que nos puede brindar algunos indicativos como estado de repositorios de git, algunas de las consolas recomendadas son:

- En windows terminal - microsoft store
- en mac y linux Hyper y se personalizar con zsh

Lista de algunos comandos basicos para consola de mac y linux y de (WSL)
Comando Descripcion Ejemplo
sudo Permite ejecutar comandos con permisos de administrador. sudo "comando"
pwd El comando muestra la ruta actual. pwd
cd Me permite navegar entre directorios. CD "ruta del directorio"
ls Lista los documentos del directorio actual, se puede usar variaciones -r lista archivos de subdirectorio, -a muestra los archivos ocultos, -l me muestra el detalle de los archivos, -t ordena la lista por fecha. ls
ls -a
ls -al
...
mkdir Crea una nueva carpeta. mkadir "nombre de la carpeta"
touch Crea un archivo. touch "Nombre del archivo"
rm Elimina archivos para eliminar carpetas se usa la varion -rf. rm "Nombre del archivo"
rm -rf "Nombre de carpeta"
cat Permite visualizar el contenido de un archivo de manera parcial cat "Nombre del archivo"
more Permite visualizar el contenido de un archivo de manera mas detallada more "Nombre del archivo"
node Permite modificar el contenido del archivo node "Nombre del archivo"
apt-get Me permite darle instrucciones al gestor de paquetes de linux como actualiza o instalar programas. sudo apt-get update
sudo apt-get upgrade
history Muestra el historial de los comandos ejecutados en la consola por medio del sombolo "!" y el numero del comando se puede repetir el comando (!246). history
!245
chown -R Da permisos de modificacion a carpetas en windows subsystem for linux. sudo chown -R "usuario" "ubicacion de la carpeta"
